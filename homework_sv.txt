AJ 1.1 Node.js {

Node.js är en server-side JavaScript runtime som möjliggör exekvering av JavaScript-kod utanför en webbläsare. Den använder V8 JavaScript runtime och följer en händelsedriven, icke-blockerande I/O-modell. Node.js är särskilt lämplig för att bygga skalbara applikationer i realtid.

Exempel:
JavaScript
// Enkel Node.js server
const http = require('http');

const server = http.createServer((req, res) => {
res.writeHead(200, {'Content-Type': 'text/plain'});
res.end('Hej, Node.js!');
});

const PORT = 3000;
server.listen(PORT, () => {
console.log(`Server som körs på http://localhost:${PORT}/`);
});

}




AJ 1.2 Express {

Express är ett minimalistiskt och flexibelt Node.js webbapplikationsramverk som ger en robust uppsättning funktioner för webb- och mobilapplikationer. Det förenklar processen att bygga skalbara och modulära webbapplikationer genom att erbjuda ett enkelt och intuitivt API.


// Simple Express-appen
const express = require('express');
const app = express();
const PORT = 3000;

app.get('/', (req, res) => {
res.send('Hej, Express!');
});

app.listen(PORT, () => {
console.log(`App-lyssning på http://localhost:${PORT}`);
});

}
